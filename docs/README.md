- **DHSR Cycle:** Differentiation → Harmonization → Syntony → Recursion
- **Q(φ) Arithmetic:** Exact golden ratio lattice mathematics
- **Syntony:** Measure of resonance quality (target: 1/φ ≈ 0.618)
- **Resonant Tensor:** Dual representation (exact lattice + ephemeral flux)
